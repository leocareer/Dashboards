### Dashboards

Collection of Power BI dashboards illustrating data modeling, transformation, and visualization techniques. Designed with attention to UX/UI best practices and data presentation quality.

### Structure

- `/Projects:` 
    - `Data.xlsx:` .xlsx data file
    - `Projects.pbix:` dashboard in pbix
    - `Projects.pdf:` dashboard in pdf

- `/Sales_KPIs:` 
    - `/Data:` .csv data files and equivalent database dump
    - `Sales_KPIs.pbix:` dashboard in pbix
    - `Sales_KPIs.pdf:` dashboard in pdf

- `/Sales_Multi-level-analysis:` 
    - `/Data:` .csv data files and equivalent database dump
    - `Sales_Multi-level-analysis.pbix:` dashboard in pbix
    - `Sales_Multi-level-analysis.pdf:` dashboard in pdf
